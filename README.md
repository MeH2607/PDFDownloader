# Welcome to the PDF downloader

<p>This app will help you download large amounts of PDF files online, and give you an overview of the downloaded files and the files that failed to download</p>

<p>This app requires you to upload an Excel file with with a list of pdf links. The file should be formated like</p>
|ID|URL|Alt URL|
|---|---|---|
|1|www.example.com|www.altexample.com|
